Partes de un Computador

Partes de un Computador

Hardware

Se denomina así a toda la parte física o tangible del medio tecnológico al cual nos estamos refiriendo, tengamos presente que no solo estamos haciendo referencia al computador, ya que nuestros dispositivos móviles, también hacen parte de las TIC’s y como tal comparten elementos en común, como son: procesador, memoria, disco duro, pantalla, teclado, mouse, cámara, etc.


Software

Es el conjunto de programas, aplicaciones y/o rutinas que le permiten al hardware realizar las diferentes funciones o tareas que el usuario le ordena al equipo. Ejemplo de ellos son los siguientes: Word, Excel, PowerPoint e inclusive los sistemas operativos de las diferentes familias, las más conocidas: Windows, Mac, Linux, entre otros

Procesador

También llamada unidad central de procesamiento (CPU) o microprocesador, es el elemento que se encarga procesar los datos e interpretar las diferentes instrucciones dadas por los programas, o sea, es el cerebro de la computadora y se encarga de gestionar todos los procesos.

Motherboard

La tarjeta madre, es la placa o tarjeta principal donde se encuentran los elementos como el procesador, las memorias, o sea, es donde se conectan todos los componentes del computador y se encarga de sincronizar el funcionamiento de estos elementos
